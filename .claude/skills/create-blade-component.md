description: Create a reusable anonymous Blade component. Delegates to frontend-specialist (sonnet).

Delegate to the `frontend-specialist` agent to create the Blade component.

Pass the user's full request. The agent will produce a complete `resources/views/components/{name}.blade.php` with @props, Tailwind classes, accessibility attributes, and all text in Portuguese-BR.
